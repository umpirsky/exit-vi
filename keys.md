### Navigation

| `j` | Go to next line |
| `k` | Go to previous line |
| `h` | Go left |
| `l` | Go right |
| `Shift` + `g` | Go to the bottom |
| `gg` | Go to top |

### Search

| `/` | Search for pattern |
| `n` | Find next occurrence |

### Commands

| `:w` | Save changes |
| `:q` | Quit |
| `:q!` | Quit without saving |
| `:wq` | Save changes and quit |
| `u` | Undo last command |

### Clipboard

| `x` | Delete character |
| `dw` | Delete word |
| `dd` | Delete line |
| `yw` | Copy word |
| `yy` | Copy line |
| `p` | Paste |
